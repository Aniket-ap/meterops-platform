# MeterOps

MeterOps is an enterprise-ready, multi-tenant SaaS platform that enables
companies to track usage, enforce rate limits, and manage subscription plans.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Redis
- Docker
- AWS

## Architecture
Modular monolith with tenant-aware middleware and clean service boundaries.
